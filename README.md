# wifi
WIFI
Normes, Sécurité et Analyse de Trames

Introduction

Le Wi-Fi est aujourd’hui incontournable dans les infrastructures réseau, que
ce soit en entreprise ou dans les environnements publics.
Derrière cette technologie se cachent de nombreuses normes, protocoles et
problématiques de sécurité.

Avant de passer à l’analyse technique, vous allez vous familiariser avec
l’univers du Wi-Fi à travers une recherche structurée.



Vous allez faire des recherches et une synthèse dans un document sur :
- Les normes Wi-Fi
- Bande de fréquence et canaux
- Architecture réseau Wi-Fi
- Sécurité des réseaux Wi-Fi
- les Normes récentes et évolutions
